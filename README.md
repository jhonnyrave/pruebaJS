# Prueba JS

En el repositorio se encuentra el desarrollo de la prueba técnica realizada en Javascript que comprende los siguientes puntos:

1. Interfaz con botones para cada uno de los ejercicios.
2. Ejercicio de clasificación de año bisiesto.
3. Creación de tablas a partir de informacion de filas y columnas.
4. Creación de arreglo aleatorio y ordenamiento.
5. Operaciones de conjuntos entre arreglos.
6. Consumo API tipo REST del Banco de MX.

## Instalación

- Clonar repositorio usando git clone https://github.com/jhonnyrave/pruebaJS.git
- Acceder al directorio pruebaJS
- Ejecutar la pagina index.html desde el navegador.
